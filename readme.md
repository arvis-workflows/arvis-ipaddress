# arvis-ipaddress

> Arvis workflow to display internal/external IP addresses

<img src="screenshot.png" width="649">

## Install

```
$ npm install --global arvis-ipaddress
```

## Usage

In arvis, type `ip`, <kbd>Enter</kbd>, and your IP addresses will be displayed.
